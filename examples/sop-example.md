# SOP Example

## Password Reset Workflow

### Objective

Provide a structured workflow for resetting enterprise user passwords.

---

## Workflow Structure

```text id="a89prx"
User Verification
      ↓
Identity Validation
      ↓
Password Reset
      ↓
MFA Verification
      ↓
Access Confirmation
```

---

## Procedure

### 1. Verify User Identity

Confirm:

* employee ID
* registered email
* manager validation if required

---

### 2. Reset Password

Perform password reset through the identity management platform.

Requirements:

* temporary password generation
* password expiration enabled
* MFA revalidation if applicable

---

### 3. Confirm Access

Validate:

* successful login
* MFA functionality
* account status

---

## Operational Notes

* document all reset actions
* escalate repeated lockouts
* avoid sharing passwords through unsecured channels
