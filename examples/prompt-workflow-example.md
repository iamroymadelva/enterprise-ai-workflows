# Prompt Workflow Example

## Scenario

Design a reusable prompt workflow for generating structured troubleshooting documentation.

---

## Workflow Structure

```text id="1wq4sd"
Workflow Goal
      ↓
Context Engineering
      ↓
Instruction Structuring
      ↓
Output Constraints
      ↓
Refinement Logic
      ↓
Reusable Prompt System
```

---

## Workflow Components

| Component         | Purpose                        |
| ----------------- | ------------------------------ |
| Context Layer     | Define operational scenario    |
| Instruction Layer | Control workflow behavior      |
| Formatting Layer  | Standardize markdown structure |
| Refinement Layer  | Improve output consistency     |

---

## Example Prompt Structure

```text id="v8rwml"
Role:
Act as a support operations documentation specialist.

Objective:
Generate a structured troubleshooting guide.

Requirements:
- concise formatting
- markdown readability
- operational clarity
- reusable workflow structure
```

---

## Operational Outcome

The workflow improves:

* prompt consistency
* reusable outputs
* workflow readability
* operational structure
